Prepare Installation Media

Setelah mengunduh citra langsung , Harus ditulis ke media yang dapat di-boot, seperti drive USB, kartu SD, atau CD/DVD.

Membuat Drive USB atau Kartu SD yang dapat di-boot di linux

Identifikasi Perangkat

Sebelum menulis citra, identifikasi perangkat tempat Anda akan menulisnya. Anda dapat melakukannya dengan cara ini. ini menggunakan fdisk. Setelah terhubung Untuk mengetahui jalur perangkat penyimpanan, jalankan perintah berikut:

# fdisk

Disk /dev/sda: 7.5 GiB, 8036286464 bytes, 15695872 sectors

Disk model: Your USB Device's Model

Units: sectors of 1 * 512 = 512 bytes

Sector size (logical/physical): 512 bytes / 512 bytes

I/O size (minimum/optimal): 512 bytes / 512 bytes

Pada contoh di atas, output menunjukkan perangkat USB sebagai /dev/sdaDi Linux, Jalur menuju perangkat biasanya berbentuk seperti ini: /dev/sdX(di mana X adalah sebuah huruf) untuk perangkat USB, /dev/mmcblkXuntuk kartu SD, atau variasi lainnya tergantung pada perangkatnya. Anda dapat menggunakan model dan ukurannya ( 7.5GiBdi atas, setelah (jalur) untuk mengidentifikasi perangkat jika Anda tidak yakin jalur mana yang akan dilaluinya.

Setelah Anda mengidentifikasi perangkat yang akan Anda gunakan, pastikan perangkat tersebut tidak terpasang dengan cara yang salah. melepaskannya dengan umount:

# umount /dev/sdX

umount: /dev/sdX: not mounted.

Membuat Live Image

Perintah dd(1) dapat digunakan untuk menyalin Mengirim citra langsung ke perangkat penyimpanan. Menggunakan dd, tulis citra langsung ke perangkat:

Warnings: Tindakan ini akan menghapus semua data yang saat ini ada di perangkat yang dirujuk. Berhati-hatilah.

# dd bs=4M if=/path/to/void-live-ARCH-DATE-VARIANT.iso of=/dev/sdX

90+0 records in

90+0 records out

377487360 bytes (377 MB, 360 MiB) copied, 0.461442 s, 818 MB/s

ddtidak akan mencetak apa pun sampai selesai (atau jika gagal), jadi, tergantung Pada perangkat, proses ini dapat memakan waktu beberapa menit atau lebih lama. Anda dapat mengaktifkan pencetakan dengan menambahkan status=progresske perintah jika menggunakan GNU coreutils dd.

Terakhir, pastikan semua data telah dihapus sebelum mencabut perangkat:

$ sync

Jumlah catatan, jumlah yang disalin, dan tarif semuanya akan bervariasi tergantung pada perangkat dan gambar langsung yang Anda pilih.

Burning/Membakar Ke CD/DVD

Aplikasi pembakaran disk apa pun harus mampu menulis .isoberkas ke CD atau DVD. Aplikasi perangkat lunak gratis berikut tersedia. (dukungan lintas platform dapat bervariasi):

  • Brasero - GNOME
  • K3b - KDE
  • Xfburn - XFCE4
  • Popsicle
    • Catatan: Perlu dicatat bahwa, dengan CD atau DVD, sesi langsung akan kurang responsif. dibandingkan dengan menggunakan USB flash drive atau hard drive.
