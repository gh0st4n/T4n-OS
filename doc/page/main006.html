Installation Guide

Setelah mengunduh citra langsung untuk instalasi dan Setelah menyiapkan media instalasi, Anda siap untuk memulai. Instal Void Linux.

Sebelum memulai instalasi, Anda harus memastikan apakah mesin Anda dapat melakukan booting. menggunakan BIOS atau UEFI. Ini akan memengaruhi cara Anda merencanakan partisi. Lihat Pemartisian Catatan untuk detail lebih lanjut.

Fitur-fitur berikut tidak didukung oleh skrip penginstal:

  • LVM
  • LUKS
  • ZFS

Booting

Nyalakan komputer Anda dari media instalasi yang telah Anda buat. Jika Anda memiliki RAM yang cukup, ada opsi di layar boot untuk memuat seluruh citra ke dalam RAM, yang Ini akan memakan waktu, tetapi akan mempercepat proses instalasi selanjutnya.

Setelah citra live berhasil di-boot, masuk sebagai rootdengan kata sandi voidlinux Dan berlari:

# void-installer

Keyboard

Pilih tata letak tombol untuk keyboard Anda; keyboard "qwerty" standar umumnya akan Gunakan peta tombol "us".

Network

Pilih antarmuka jaringan utama Anda. Jika Anda tidak memilih untuk menggunakan DHCP, Anda Anda akan diminta untuk memberikan alamat IP, gateway, dan server DNS.

Jika Anda memilih antarmuka jaringan nirkabel, Anda akan diminta untuk memberikan informasi berikut: SSID, jenis enkripsi ( wpa atau wep), dan kata sandi. Jika void-installergagal Untuk terhubung ke jaringan Anda, Anda mungkin perlu keluar dari penginstal dan mengkonfigurasinya. secara manual menggunakan wpa_supplicant dan Periksa dhcpcd sebelum melanjutkan.

Source

Untuk menginstal paket yang disediakan pada image instalasi, pilih Local. Jika tidak, Anda dapat memilih Networkuntuk mengunduh paket terbaru dari Void gudang.

Warnings: Jika Anda menginstal lingkungan desktop dari xfce gambar, Anda HARUS memilih Localuntuk sumbernya!
