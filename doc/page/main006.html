Installation Guide

Setelah mengunduh citra langsung untuk instalasi dan Setelah menyiapkan media instalasi, Anda siap untuk memulai. Instal Void Linux.

Sebelum memulai instalasi, Anda harus memastikan apakah mesin Anda dapat melakukan booting. menggunakan BIOS atau UEFI. Ini akan memengaruhi cara Anda merencanakan partisi. Lihat Pemartisian Catatan untuk detail lebih lanjut.

Fitur-fitur berikut tidak didukung oleh skrip penginstal:

  • LVM
  • LUKS
  • ZFS

Booting

Nyalakan komputer Anda dari media instalasi yang telah Anda buat. Jika Anda memiliki RAM yang cukup, ada opsi di layar boot untuk memuat seluruh citra ke dalam RAM, yang Ini akan memakan waktu, tetapi akan mempercepat proses instalasi selanjutnya.

Setelah citra live berhasil di-boot, masuk sebagai rootdengan kata sandi voidlinux Dan berlari:

# void-installer

Keyboard

Pilih tata letak tombol untuk keyboard Anda; keyboard "qwerty" standar umumnya akan Gunakan peta tombol "us".

Network

Pilih antarmuka jaringan utama Anda. Jika Anda tidak memilih untuk menggunakan DHCP, Anda Anda akan diminta untuk memberikan alamat IP, gateway, dan server DNS.

Jika Anda memilih antarmuka jaringan nirkabel, Anda akan diminta untuk memberikan informasi berikut: SSID, jenis enkripsi ( wpa atau wep), dan kata sandi. Jika void-installergagal Untuk terhubung ke jaringan Anda, Anda mungkin perlu keluar dari penginstal dan mengkonfigurasinya. secara manual menggunakan wpa_supplicant dan Periksa dhcpcd sebelum melanjutkan.

Source

Untuk menginstal paket yang disediakan pada image instalasi, pilih Local. Jika tidak, Anda dapat memilih Networkuntuk mengunduh paket terbaru dari Void gudang.

Warnings: Jika Anda menginstal lingkungan desktop dari xfce gambar, Anda HARUS memilih Localuntuk sumbernya!

Hostname

Membuat Nama Host untuk Komputer

Local

Pilih pengaturan lokal default Anda. Opsi ini hanya untuk glibc, seperti halnya musl. Saat ini belum mendukung beberapa wilayah.

Zona Waktu

Pilih zona waktu Anda berdasarkan opsi zona waktu standar.

Password Root

Masukkan dan konfirmasikan rootKata sandi untuk instalasi baru. Kata sandi tidak akan ditampilkan di layar.

User Account

Pilih login (default) "void" dan nama deskriptif untuk login tersebut. Kemudian Masukkan dan konfirmasikan kata sandi untuk pengguna baru. Anda kemudian akan diminta untuk Verifikasi grup untuk pengguna baru ini. Mereka ditambahkan ke wheelkelompokkan berdasarkan default dan akan memiliki sudo akses. Grup default dan deskripsinya adalah tercantum di sini.

Nama pengguna memiliki beberapa batasan, seperti yang dijelaskan dalam useradd .

Bootloader

Pilih disk tempat bootloader akan diinstal saat Void diinstal. Anda dapat memilih noneuntuk melewati langkah ini dan menginstal bootloader secara manual setelahnya menyelesaikan proses instalasi. Jika menginstal bootloader, Anda juga akan Anda akan ditanya apakah Anda menginginkan terminal grafis untuk menu GRUB atau tidak.

Partition

Selanjutnya, Anda perlu mempartisi disk Anda. Void tidak menyediakan partisi yang sudah diatur sebelumnya. skema partisi, jadi Anda perlu membuat partisi secara manual dengan cfdisk(8) . Anda akan diminta dengan Daftar disk. Pilih disk yang ingin Anda partisi dan penginstal akan meluncurkan cfdiskuntuk disk tersebut. Ingat, Anda harus menulis tabel partisi ke drive tersebut sebelum Anda keluar dari editor partisi.

Jika menggunakan UEFI, disarankan Anda memilih GPT untuk tabel partisi dan Buat partisi (biasanya antara 200MB-1GB) dengan tipe EFI System, yang akan dipasang di /boot/efi.

Jika menggunakan BIOS, disarankan untuk memilih MBR untuk tabel partisi. Pengguna tingkat lanjut dapat menggunakan GPT tetapi perlu membuat partisi BIOS khusus untuk GRUB agar dapat melakukan booting.

Sistem file root membutuhkan partisi bertipe Linux filesystemyang akan menjadi dipasang di /.

Lihat Note Partition untuk detail selengkapnya tentang Mempartisi disk Anda.

File System

Buat sistem file untuk setiap partisi yang telah Anda buat. Untuk setiap partisi Anda akan diminta untuk memilih jenis sistem file, apakah Anda ingin membuat sistem file baru pada partisi, dan titik pemasangan, jika berlaku. Saat Anda selesai, pilih Doneuntuk kembali ke menu utama.

Jika menggunakan UEFI, buatlah vfatsistem file dan pasang di /boot/efi.

Review Settings

Sebaiknya periksa pengaturan Anda sebelum melanjutkan. Gunakan panah kanan. Tekan tombol untuk memilih tombol pengaturan dan tekan Enter Semua pilihan Anda akan ditampilkan untuk ditinjau.

Installation

Memilih Install Dari menu akan memulai penginstal. Penginstal akan Buat semua sistem file yang dipilih, dan instal paket sistem dasar. Ini Kemudian akan menghasilkan initramfs dan menginstal bootloader GRUB2 ke perangkat yang dapat di-boot. partisi.

Semua langkah ini akan berjalan secara otomatis, dan setelah instalasi selesai. Berhasil, Anda dapat melakukan reboot ke instalasi Void Linux baru Anda!

Post Installation

Setelah melakukan booting ke instalasi Void Anda untuk pertama kalinya, lakukan pemeriksaan sistem dan pembaruan.
