Users and Groups

useradd, userdel dan usermod digunakan untuk menambahkan, menghapus dan memodifikasi pengguna secara berturut-turut. Perintah passwd digunakan untuk mengubah kata sandi.

groupadd, grupdel dan groupmod digunakan untuk menambahkan, menghapus dan memodifikasi grup secara berturut-turut. The Perintah groups mencantumkan semua grup yang dimiliki pengguna. milik.

Shell Default

Shell default untuk seorang pengguna dapat diubah dengan chsh :

$ chsh -s [shell] [username]

[shell] harus berupa jalur ke shell seperti yang ditentukan oleh /etc/shells atau keluaran dari chsh -l, yang menyediakan daftar shell yang terpasang.

Sudo

Sudo diinstal secara default, tetapi mungkin mungkin tidak dikonfigurasi dengan tepat untuk kebutuhan Anda. Hanya perlu Konfigurasikan sudo jika Anda ingin menggunakannya.

Gunakan visudo sebagai root untuk mengedit sudoers file.

Untuk membuat superuser, hapus tanda komentar pada baris tersebut. 

#%wheel ALL=(ALL) ALL

dan menambahkan pengguna ke wheel kelompok.

Group Default

Void Linux mendefinisikan sejumlah grup secara default.

daemon Daemon sistem yang perlu menulis ke file di disk.
floppy Akses ke drive floppy.
disk Akses mentah ke /dev/sd* Dan /dev/loop*.
lp Akses ke printer
dialout Akses ke drive port serial.
audio Akses ke perangkat audio.
video Akses ke perangkat video.
utmp Kemampuan untuk menulis kepada /var/run/utmp, /var/log/wtmp Dan /var/log/btmp.
adm Tidak digunakan - ada karena alasan historis. Grup ini secara tradisional digunakan untuk pemantauan sistem, seperti melihat file di /var/log.
cdrom Akses ke CD.
optical Akses ke perangkat DVD/CD-RW.
mail Digunakan oleh beberapa paket pos, misalnya dma.
storage Akses ke perangkat penyimpanan yang dapat dilepas.
scanner Kemampuan mengakses pemindai.
network Digunakan oleh beberapa paket yang berhubungan dengan jaringan, misalnya connman, NetworkManager, wicd.
kvm Kemampuan untuk menggunakan KVM untuk mesin virtual, misalnya melalui QEMU.
input Akses ke perangkat input: /dev/mouse*, /dev/event*.
plugdev Akses ke perangkatyang dapat dicolokkan
nogroup Deamon sistem yang tidakperlu memiliki file apa pun
usbmon Akses ke /dev/usbmon*.
users Pengguna biasa
xbuilder Untuk menggunakan xbps-uchroot dengan xbps-src.
