Installer Images

T4n OS merilis enam jenis image: image base runit, image base systemd, image xfce4, image bspwm, image kde dan image riverwm. Untuk pemula Linux. Disarankan untuk mencoba salah satu image XFCE/KDE yang lebih lengkap fiturnya, tetapi lebih Pengguna tingkat lanjut mungkin sering kali lebih memilih untuk memulai dari image dasar untuk hanya menginstal hal-hal tertentu. paket yang mereka butuhkan.

image base runit

Image base runit hanya menyediakan serangkaian paket minimal untuk menginstal Void yang dapat digunakan. sistem. Paket dasar ini hanya mencakup paket yang dibutuhkan untuk mengkonfigurasi mesin baru, Perbarui sistem, runit dan instal paket tambahan dari repositori.

image base systemd

image base systemd hanya menyediakan serangkaian paket minimal untuk menginstal Void yang dapat digunakan. sistem. Paket dasar ini hanya mencakup paket yang dibutuhkan untuk mengkonfigurasi mesin baru, Perbarui sistem, systemd dan instal paket tambahan dari repositori.

image XFCE

Citra XFCE menyertakan lingkungan desktop lengkap, peramban web, dan fitur dasar. aplikasi yang dikonfigurasi untuk lingkungan tersebut. Satu-satunya perbedaan dari basis Gambar-gambar tersebut adalah paket dan layanan tambahan yang diinstal.

Perangkat lunak berikut disertakan:

  • Window Manager : xfwm4
  • File Manager : Thunar
  • Browser : Firefox
  • Terminal : xfce4-terminal
  • Plain Text Editor : mousepad
  • Image Viewer : Ristretto
  • Other : Bulk rename, Orage Globaltime, Orage Calendar, Task Manager, Parole Media Player, Audio Mixer, MIME type editor, Application finder
