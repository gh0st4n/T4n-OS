Tentang T4n OS

T4n OS adalah sistem operasi Linux berbasis Void Linux, dirancang untuk menghadirkan kekuatan level power user dengan kenyamanan untuk general user, tanpa mengorbankan kendali pengguna atas sistem.

Berbeda dengan distro lain yang mengambil keputusan atas nama user, T4n OS memberi default yang masuk akal, tapi keputusan akhir tetap di tangan user.

T4n OS adalah Project Mandiri (Gh0sT4n) - Bocah Atjeh/Siswa Darussa'adah Cot Tarom Baroh - SMKN 5 Takengon
