Selamat Datang di T4n OS

T4n OS adalah distribusi Linux yang ringan dan cepat, berbasis Void Linux dengan fokus pada stabilitas, keamanan, dan kemudahan penggunaan. T4n OS dirancang untuk pengguna yang menginginkan sistem operasi yang responsif dengan kontrol penuh atas lingkungan mereka.

Catatan: T4n OS menggunakan manajer paket XBPS (X Binary Package System) yang sangat cepat dan efisien.

Mengapa Memilih T4n OS?

Performa Tinggi

T4n OS dioptimalkan untuk performa tinggi dengan boot time yang cepat dan penggunaan sumber daya yang minimal.

Rolling Release

Dengan model rolling release, Anda selalu mendapatkan pembaruan terbaru tanpa perlu melakukan upgrade versi mayor.

Keamanan

T4n OS memiliki fokus pada keamanan dengan pembaruan keamanan yang cepat dan default yang aman.

Fleksibilitas

Dengan berbagai pilihan lingkungan desktop dan aplikasi, Anda dapat menyesuaikan T4n OS sesuai kebutuhan Anda.

Instalasi T4n OS

Berikut adalah panduan langkah demi langkah untuk menginstal T4n OS pada sistem Anda:

1. Persiapan

  • Unduh ISO T4n OS dari situs resmi
  • Buat bootable USB dengan menggunakan alat seperti Rufus (Windows) atau dd (Linux)
  • Backup data penting Anda
  • pastikan sistem memenuhi persyaratan minimum:
    • Prosesor: x86_64
    • RAM: minimal 1GB (direkomendasikan 2GB)
    • Penyimpanan: minimal 10GB

2. Proses Instalasi

  1. Boot dari USB yang telah Anda siapkan
  2. Pilih "Start T4n OS" dari menu boot
  3. Buka terminal dan jalankan perintah berikut untuk memulai instalasi:
sudo t4n-installer
  1. Ikuti panduan di layar untuk mengkonfigurasi:
    • Bahasa dan zona waktu
    • Partisi disk
    • Pengguna dan password
    • Lingkungan desktop (opsional)
  2. Tunggu hingga proses instalasi selesai
  3. Restart sistem dan lepaskan USB
Peringatan: Pastikan Anda memilih partisi yang benar saat menginstal untuk menghindari kehilangan data.

Manajemen Paket

T4n OS menggunakan XBPS (X Binary Package System) sebagai manajer paketnya. XBPS adalah manajer paket yang cepat, efisien, dan andal.

Perintah Dasar XBPS

Memperbarui Repository

sudo xbps-install -S

Menginstal Paket

sudo xbps-install nama_paket

Menghapus Paket

sudo xbps-remove nama_paket

Mencari Paket

xbps-query -Rs kata_kunci

Informasi Paket

xbps-query -R nama_paket

Memperbarui Sistem

sudo xbps-install -Su

Repository Tambahan

T4n OS memiliki beberapa repository yang dapat Anda tambahkan:

  • Nonfree: Untuk perangkat lunak non-libre
  • Multilib: Untuk aplikasi 32-bit di sistem 64-bit
  • Community: Untuk paket yang dipelihara oleh komunitas

Untuk menambahkan repository, edit file /etc/xbps.d/00-repository-main.conf dan tambahkan baris berikut:

repository=https://repo-default.voidlinux.org/current/nonfree
repository=https://repo-default.voidlinux.org/current/multilib
repository=https://repo-default.voidlinux.org/current/community

Lingkungan Desktop

T4n OS mendukung berbagai lingkungan desktop yang dapat Anda pilih sesuai preferensi:

KDE Plasma

Lingkungan desktop modern dengan fitur lengkap dan kustomisasi tinggi.

sudo xbps-install -S kde5

XFCE

Lingkungan desktop ringan namun powerful, ideal untuk sistem dengan sumber daya terbatas.

sudo xbps-install -S xfce4

LXQt

Lingkungan desktop yang sangat ringan dan modern, dibangun dengan Qt.

sudo xbps-install -S lxqt

MATE

Lingkungan desktop yang melanjutkan tradisi GNOME 2, stabil dan familiar.

sudo xbps-install -S mate

Cinnamon

Lingkungan desktop yang dikembangkan oleh Linux Mint, modern dan intuitif.

sudo xbps-install -S cinnamon

i3wm

Manajer jendela tiling yang ringan dan efisien untuk pengguna produktivitas.

sudo xbps-install -S i3

Mengaktifkan Display Manager

Setelah menginstal lingkungan desktop, Anda perlu menginstal dan mengaktifkan display manager:

# Untuk SDDM (direkomendasikan untuk KDE)
sudo xbps-install sddm
sudo ln -s /etc/sv/sddm /var/service/

# Untuk LightDM (kompatibel dengan sebagian besar DE)
sudo xbps-install lightdm lightdm-gtk3-greeter
sudo ln -s /etc/sv/lightdm /var/service/

Konfigurasi Jaringan

T4n OS menggunakan NetworkManager untuk mengelola koneksi jaringan. Berikut adalah panduan konfigurasi jaringan dasar:

1. Menggunakan NetworkManager

NetworkManager biasanya sudah terinstal dan berjalan secara default. Anda dapat menggunakannya melalui:

  • GUI: nm-applet (applet di system tray)
  • CLI: nmcli
  • TUI: nmtui

2. Menghubungkan ke WiFi via CLI

# Melihat jaringan WiFi yang tersedia
nmcli dev wifi list

# Menghubungkan ke jaringan WiFi
nmcli dev wifi connect "SSID" password "password_anda"

3. Konfigurasi IP Statis

# Mengatur IP statis untuk koneksi kabel
nmcli con mod "Wired connection 1" ipv4.addresses 192.168.1.100/24
nmcli con mod "Wired connection 1" ipv4.gateway 192.168.1.1
nmcli con mod "Wired connection 1" ipv4.dns "8.8.8.8,8.8.4.4"
nmcli con mod "Wired connection 1" ipv4.method manual
nmcli con up "Wired connection 1"

4. Firewall

T4n OS menggunakan iptables/nftables sebagai firewall default. Untuk kemudahan, Anda dapat menggunakan ufw (Uncomplicated Firewall):

# Instalasi
sudo xbps-install ufw

# Mengaktifkan firewall
sudo ufw enable

# Mengizinkan SSH
sudo ufw allow ssh

# Melihat status
sudo ufw status

Keamanan

Keamanan adalah prioritas utama dalam T4n OS. Berikut adalah beberapa praktik keamanan yang direkomendasikan:

1. Pembaruan Keamanan

Secara berkala periksa dan instal pembaruan keamanan:

sudo xbps-install -Su

2. Pengguna Non-Root

Hindari menggunakan akun root untuk aktivitas sehari-hari. Gunakan sudo untuk perintah yang memerlukan hak akses root:

# Menambahkan pengguna ke grup wheel
sudo usermod -aG wheel nama_pengguna

3. Enkripsi Disk

Untuk melindungi data Anda, pertimbangkan untuk mengenkripsi disk dengan LUKS:

# Membuat partisi terenkripsi
sudo cryptsetup luksFormat /dev/sdXn

# Membuka partisi terenkripsi
sudo cryptsetup open /dev/sdXn cryptname

# Membuat filesystem
sudo mkfs.ext4 /dev/mapper/cryptname

4. AppArmor

AppArmor adalah modul keamanan kernel yang membatasi kemampuan aplikasi:

# Instalasi
sudo xbps-install apparmor

# Mengaktifkan profil
sudo aa-enforce /etc/apparmor.d/*
Tip: Gunakan kata sandi yang kuat dan unik untuk setiap akun. Pertimbangkan untuk menggunakan pengelola kata sandi seperti KeePassXC.

Pemecahan Masalah

Berikut adalah solusi untuk beberapa masalah umum yang mungkin Anda temui di T4n OS:

1. Sistem Tidak Boot

Jika sistem tidak dapat boot, coba langkah-langkah berikut:

  • Boot dari live USB dan gunakan chroot untuk memperbaiki masalah:
# Mount partisi root
sudo mount /dev/sdXn /mnt

# Mount partisi boot jika terpisah
sudo mount /dev/sdYn /mnt/boot

# Mount filesystem virtual
sudo mount --bind /dev /mnt/dev
sudo mount --bind /proc /mnt/proc
sudo mount --bind /sys /mnt/sys

# Masuk ke sistem yang terinstal
sudo chroot /mnt /bin/bash

2. Masalah Jaringan

Jika Anda mengalami masalah dengan koneksi jaringan:

  • Periksa status NetworkManager:
sudo sv status NetworkManager
  • Restart NetworkManager jika diperlukan:
sudo sv restart NetworkManager

3. Masalah Audio

Jika audio tidak berfungsi:

  • Periksa apakah ALSA terinstal:
sudo xbps-install alsa-utils
  • Unmute channel audio:
amixer sset Master unmute
amixer sset Speaker unmute

4. Masalah Display Manager

Jika display manager tidak berfungsi:

  • Periksa log:
journalctl -b -u display-manager
  • Restart display manager:
sudo sv restart display-manager

5. Masalah Paket Rusak

Jika Anda mengalami masalah dengan paket yang rusak:

# Perbaiki database paket
sudo xbps-install -Sf

# Hapus cache paket yang rusak
sudo xbps-remove -O

FAQ (Pertanyaan yang Sering Diajukan)

Q: Apa perbedaan antara T4n OS dan Void Linux?

A: T4n OS adalah distribusi berbasis Void Linux dengan tambahan repositori, tema khusus, dan beberapa aplikasi pre-installed untuk pengalaman yang lebih mudah bagi pengguna.

Q: Bagaimana cara mengupgrade T4n OS?

A: T4n OS menggunakan model rolling release, jadi Anda hanya perlu menjalankan sudo xbps-install -Su secara berkala untuk mendapatkan pembaruan terbaru.

Q: Bisakah saya menggunakan T4n OS di laptop lama?

A: Ya, T4n OS dirancang untuk ringan dan efisien, sehingga dapat berjalan dengan baik di laptop lama. Pilih lingkungan desktop yang ringan seperti XFCE atau LXQt untuk performa terbaik.

Q: Bagaimana cara menginstal perangkat lunak non-libre?

A: Anda perlu menambahkan repository nonfree dengan mengedit file /etc/xbps.d/00-repository-main.conf dan menambahkan baris repository=https://repo-default.voidlinux.org/current/nonfree, lalu jalankan sudo xbps-install -S.

Q: Bagaimana cara mengoptimalkan baterai laptop?

A: Instal tlp untuk manajemen daya yang lebih baik:

sudo xbps-install tlp
sudo ln -s /etc/sv/tlp /var/service/
sudo sv up tlp

Q: Bisakah saya menjalankan aplikasi Windows di T4n OS?

A: Ya, Anda dapat menggunakan Wine untuk menjalankan banyak aplikasi Windows:

sudo xbps-install wine

Q: Bagaimana cara berkontribusi pada T4n OS?

A: Anda dapat berkontribusi dengan melaporkan bug, menerjemahkan dokumentasi, atau mengembangkan paket. Kunjungi situs resmi T4n OS untuk informasi lebih lanjut.
