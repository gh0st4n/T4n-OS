Tentang T4n OS

T4n OS

T4n OS adalah distribusi Linux yang berbasis Void Linux dengan fokus pada kustomisasi dan performa. Sistem ini dirancang untuk pengguna yang menginginkan kontrol penuh atas sistem operasi mereka.

Kunjungi GitHub

VUR (Void User Repo)

VUR adalah repositori pengguna untuk T4n OS yang mirip dengan AUR di Arch Linux. Memungkinkan pengguna untuk berbagi dan menginstal perangkat lunak tambahan yang tidak tersedia di repositori resmi Void Linux.

Kunjungi GitHub

VUR-Helper

Tools Python Wrapper yang mengambil source code, mengompilasi, dan menggunakan tools XDEB dan xbps-src untuk mempermudah proses pembuatan paket untuk VUR.

Kunjungi GitHub

Fitur T4n OS

CLI/Server

Optimasi untuk penggunaan CLI dan server dengan performa tinggi dan jejak memori minimal.

GUI X11 - XFCE4

Dukungan penuh untuk XFCE4 sebagai lingkungan desktop X11 yang ringan dan responsif.

GUI Wayland - COSMIC DE

Support untuk COSMIC Desktop Environment pada protokol Wayland yang modern.

WM X11 - BSPWM

Konfigurasi BSPWM yang optimal untuk produktivitas dengan manajemen jendela tiling.

WM Wayland - RiverWM

Dukungan untuk RiverWM atau WalaweWM (Wayland Compositor) sebagai alternatif modern.

Kustomisasi Init System

Sistem init runit yang dikustomisasi untuk booting cepat dan manajemen layanan yang efisien.

Komponen Inti

XDEB

Tools untuk mengkonversi paket dari berbagai distribusi ke format XBPS yang digunakan oleh Void Linux dan T4n OS.

xbps-src

Tools build dari Void Linux yang digunakan untuk membuat paket dari source code untuk repositori Void Linux dan VUR.

runit
