Tentang T4n OS

T4n OS

T4n OS adalah Distribusi Linux yang berbasis Void Linux. Sistem ini dirancang untuk pengguna yang menginginkan kontrol penuh(Power User) dan pengguna umum seperti : Browsing, Office, Content Creator, Editor, Programmer dan Server

Kunjungi GitHub

VUR (Void User Repository)

VUR atau Void User Repository adalah repository dengan kumpulan template yang berguna seperti AUR dan menutupi kekurangan repository resmi void linux

Highly Customizable

Lightweight

Pilihan Sistem Init

Fitur

VUR Integration COMING SOON

Integrasi penuh dengan Void User Repository (VUR) yang menyediakan akses ke ribuan paket tambahan yang tidak tersedia di repository resmi Void Linux.

Multiple Desktop Environments

Dukungan untuk berbagai lingkungan desktop termasuk XFCE4 untuk X11 dan KDE untuk Wayland, memberikan fleksibilitas penuh kepada pengguna.

Optimized Performance

Optimasi khusus untuk berbagai use case dari daily computing hingga server production, dengan fokus pada penggunaan sumber daya yang minimal.

Custom Init System COMING SOON

Sistem init runit yang dikustomisasi untuk booting ultra-cepat dan manajemen layanan yang efisien, dengan opsi untuk menggunakan SystemD pada server.
